Using Angular you can create your own view filters that format the data in a way that is not available in the out-of-box angular filters.
Value of the model variables without any filter.
{{customer.firstName}}
{{customer.lastName}}
{{customer.accountType}}
{{customer.accountBalance}}
Value of the model variables with filters applied.
{{customer.firstName | capitalize}}
{{customer.lastName | capitalize}}
{{customer.accountType | lowercase}}
{{customer.accountBalance | currency}}
