Problem with binding to simple types in a controller. Try changing the value in the Input Field for in the child Controller. That creates a new variable inside the child controller scope and hence breaks the bindings between the parent and child.

Parent Controller

Parent 1 Name : {{name}}

Child Controller

Child 1 Name : {{name}}
By putting your simple variable inside an object you are forcing angular not to create a new scope variable inside the child controller if it does not find the model there.
