You can use the "number" filter to display numbers (Integer and floats) in a format that confirms to the user locale. It can also round off the number to as many decimal places that you want.
Using a "json" filter you can display raw JSON data in your controller on the view.
Number Filter
{{users.num}}
{{users.num | number}}
{{users.num | number:2}}
{{users.num | number:0}}
JSON Filter 
          {{users.userList | json:2}}
