Using the Angular view filters you can format the value on the "view" without changing the data on the model. This allows to keep a clear separation of concern between the model and view.
Value of the model variables without any filter.
{{customer.firstName}}
{{customer.lastName}}
{{customer.accountType}}
{{customer.accountBalance}}
Value of the model variables with filters applied.
{{customer.firstName | uppercase}}
{{customer.lastName | uppercase}}
{{customer.accountType | lowercase}}
{{customer.accountBalance | currency}}
